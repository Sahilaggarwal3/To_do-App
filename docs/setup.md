# Setup Guide — TaskMaster To-Do App

Welcome! Before we dive into coding, let’s get your local environment ready so **TaskMaster** can run smoothly.

---

## STEP-1 : Clone the Repository

Open a terminal (or VS Code terminal) and run:

```bash
git clone https://github.com/Sahilaggarwal3/To_do-App.git
cd To_do-App

## STEP-2 : Install Dependencies

The project uses Node.js and Express, so install required packages:
npm install

Dependencies installed:

express → Handles server routes

body-parser → Parses incoming request bodies

cors → Allows cross-origin requests

## STEP-3 : Start the Application
npm start

This starts the server (server.js) and serves the frontend from public/.

## STEP - 4: Open in Browser

Go to:
http://localhost:3000

You should now see the TaskMaster homepage!

## STEP - 5: Notes

Node.js v14+ recommended

Keep terminal open while using the app

Check installed dependencies if errors occur:

npm list
